# Game Development Experience

## Overview

Zohaib is a game developer with hands-on experience building games using modern game engines and tools. He enjoys working on all aspects of game development — from core gameplay mechanics to level design, UI/UX, and performance optimization.

## Game Engines & Tools

- **Unity** — Primary engine for 2D and 3D games. Experienced with C# scripting, physics, animation systems, and the Unity UI toolkit.
- **Unreal Engine** — Used for more graphically intensive 3D projects. Familiar with Blueprints and C++ integrations.
- **Godot** — Enjoys Godot for rapid prototyping and 2D game development. Appreciates the open-source community.

## Programming Languages

- C# (Unity)
- C++ (Unreal Engine, low-level game systems)
- GDScript (Godot)
- JavaScript / TypeScript (web-based games and tooling)

## Areas of Interest

## Game Design Philosophy

Zohaib believes great games emerge from the intersection of tight core mechanics, compelling feedback loops, and a clear design vision. He is influenced by game designers like Shigeru Miyamoto, Hidetaka Miyazaki, and Eric Barone (ConcernedApe), each of whom built games with a strong sense of identity and purpose.
