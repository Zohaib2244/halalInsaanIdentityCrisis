# Knowledge Base Examples

This folder contains **example markdown files** showing what kind of content to upload to Zohaib's chatbot knowledge base via the `/dashboard`.

## How to Use

1. Run the site locally with `npm run dev`
2. Go to `http://localhost:4321/dashboard` and log in with your `DASHBOARD_PASSWORD`
3. Copy the content from these example files (or paste your own real content) into the dashboard uploader
4. The chatbot will immediately start using the uploaded content to answer questions

## Files

| File | What to put here |
|------|-----------------|
| `about-zohaib.md` | Bio, background, contact info |
| `favorite-games.md` | Favorite games, currently playing, gaming philosophy |
| `game-dev-experience.md` | Engines, tools, projects, skills, design philosophy |

## Tips

- **Be specific**: "I love Hollow Knight because of its precise controls and oppressive atmosphere" is more useful than "I like action games"
- **Paste as separate documents**: Upload each file separately so the retrieval system can find the most relevant chunks
- **Update regularly**: The more you add, the smarter the bot gets — add new games you finish, new projects you ship, etc.
- **Real data beats templates**: Replace all the placeholder content with your actual experience and preferences
